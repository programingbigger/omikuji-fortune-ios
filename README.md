# omikuji-fortune-ios

🎋 A simple omikuji (Japanese fortune-telling) app built with SwiftUI.

---

## English

### Overview
Omikuji is a traditional Japanese fortune-telling practice.  
This app lets you draw a digital omikuji and discover your fortune.

### Tech Stack
- Swift
- SwiftUI
- Xcode 16+
- iOS 17+

### Status
Work in progress — built for personal learning purposes.
